Wargame: Krypton

Level: 3

Category: Cryptography

Description

The password is encrypted with a simple substitution cipher (each plaintext letter maps to a fixed ciphertext letter). Several ciphertext found files of known-genre text are available alongside the target ciphertext. The goal is to recover the substitution alphabet.

Overview

  • A monoalphabetic substitution cipher permutes the alphabet; frequency analysis exploits the fact that letter frequencies in English are well-known.
  • We concatenate all found files to build a large ciphertext corpus, then count letter frequencies.
  • The most frequent ciphertext letter is mapped to E, the second to A, and so on following the English frequency table.
  • Applying the recovered substitution to the target ciphertext reveals the password.

Background: Frequency Analysis

  • In English text, the most common letters in order are approximately: E, A, T, S, O, R, I, N, H, C, …
  • A substitution cipher preserves letter frequencies; only the labels change.
  • Given enough ciphertext, frequency matching gives a close initial mapping that can be refined manually.
  • This attack fails against polyalphabetic ciphers (Vigenere, OTP) because each plaintext letter can map to different ciphertext letters.

Solution

Step 1: Build the Frequency Table

  • Read all found* files in /krypton/krypton3/ and concatenate their contents.
  • Count the frequency of each letter using collections.Counter.
from collections import Counter

total = []
for f in found_files:
    content = ssh.system(f"cat {f}").recv().decode().strip()
    total += [c for c in content if c.isalpha()]

freq = Counter(total).most_common()
cipher_order = [t[0] for t in freq]

Step 2: Build the Translation Table

  • Map each ciphertext letter (sorted by frequency) to the corresponding English frequency letter.
eng_freq = ["E", "A", "T", "S", "O", "R", "I", "N", "H", "C",
            "L", "D", "U", "P", "M", "F", "W", "G", "Y", "B",
            "K", "V", "X", "Q", "J", "Z"]

table = str.maketrans("".join(cipher_order), "".join(eng_freq))
plaintext = cipher.translate(table)

Step 3: Verify Against SSH Login

  • Split the plaintext on spaces; each word is a candidate password.
  • Try each candidate as the krypton4 SSH password until one succeeds.
