Setup

Definition:

Continuous Contribution Public Good Game

  • Players: N>2N>2N>2 players.
  • Strategies: Each player iii chooses a contribution xi∈[0,5]x_i \in [0,5]xi​∈[0,5].

Rules

  • Start with each player endowed with 555 units; Players simultaneously choose contributions xi∈[0,5]x_i\in[0,5]xi​∈[0,5] to a common pool.
  • The player who chooses the payoff-maximising contribution against others’ contributions is optimal.
  • Each player is endowed with 555 units of a private asset; Contributions are added to a common pool.
  • Payoffs are realised simultaneously after all contributions are chosen.

Derivation (Best Response Analysis)

  • Holding other contributions fixed, player iii‘s problem is
max⁡xi∈[0,5](10−xi+∑k≠ixk).\max_{x_i \in [0,5]} \left(10 - x_i + \sum_{k\neq i} x_k \right).xi​∈[0,5]max​​10−xi​+k=i∑​xk​​.

Derivation (Nash Equilibrium)

  • Since every player’s best response is 000, the unique Nash equilibrium is

  • To find the social optimum, maximise total welfare:
W=∑i=1Nπi=10N+(N−2)∑i=1Nxi.W = \sum_{i=1}^{N} \pi_i = 10N + (N-2)\sum_{i=1}^{N} x_i.W=i=1∑N​πi​=10N+(N−2)i=1∑N​xi​.

Nash Equilibrium

Result:

The unique Nash equilibrium is zero contribution by every player:

(0,0,…,0).(0,0,\dots,0).(0,0,…,0).

Social Optimum

  • The welfare-maximising outcome is full contribution by every player:
(5,5,…,5).(5,5,\dots,5).(5,5,…,5).
  • The gap between equilibrium and optimum is the free-rider problem.

Insights

Insight:

  • Each player bears the full private cost of contribution but captures only part of the social benefit.
  • This is the continuous-strategy analogue of a Prisoner’s Dilemma.
