Game theory analysis: Sequential Take-Away Game.

1 May 2026
  • Content map: SMU H3 Game Theory Map

Setup

Definition:

Sequential Take-Away Game

  • Players: Player 1 and Player 2.
  • Strategies: On each turn, remove 1, 2, or 3 objects.
  • Rules: Start with 212121 objects; players alternate moves and the player who removes the last object wins.

Derivation (Backward Induction)

  • Starting from the end state of 1,2,31,2,31,2,3, the player is winning for the player.
  • The positions 0,4,8,12,16,200,4,8,12,16,200,4,8,12,16,20 are losing for the player.
  • From any non-multiple of 444, a player can move to the nearest lower multiple of 444.

Winning Strategy

  • First move: remove 111 to leave 202020.
  • Thereafter: if the opponent removes x∈{1,2,3}x\in\{1,2,3\}x∈{1,2,3}, remove 4−x4-x4−x to restore a multiple of 444.

Nash Equilibrium

Result:

The first player has a winning strategy: always leave a multiple of 444 to the opponent.

Insights

Insight:

The game is solved by identifying a winning invariant (here: leaving multiples of 444). This is a standard backward-induction pattern in take-away games.
