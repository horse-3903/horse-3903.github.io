Setup

Definition:

Mixed-Strategy Anti-Coordination Game

  • Players: nnn identical players.
  • Strategies: Each player chooses Blonde or Brunette.

Rules

  • Start with a group choosing between Blonde and Brunette; Players simultaneously choose one target.
  • The player who is alone on Blonde, or avoids crowding on Blonde, gets the preferred outcome.
  • Choices are made simultaneously.
  • Payoffs satisfy a>b>c>da>b>c>da>b>c>d, with the notes using a=3a=3a=3, b=2b=2b=2, c=1c=1c=1, d=0d=0d=0; A player choosing Blonde gets aaa if alone on Blonde and ddd if at least one other player also chooses Blonde.
  • A player choosing Brunette gets ccc if exactly one other player chooses Blonde and bbb otherwise.

Mixed Strategy Derivation

  • Let ppp be the probability that each of the other n−1n-1n−1 players chooses Blonde.
  • Expected payoff from choosing Blonde:
E[π(Blonde)]=3(1−p)n−1.E[\pi(\text{Blonde})] = 3(1-p)^{n-1}.E[π(Blonde)]=3(1−p)n−1.
  • Expected payoff from choosing Brunette:
E[π(Brunette)]=(n−1)p(1−p)n−2+2[1−(n−1)p(1−p)n−2].E[\pi(\text{Brunette})] = (n-1)p(1-p)^{n-2} + 2\left[1-(n-1)p(1-p)^{n-2}\right].E[π(Brunette)]=(n−1)p(1−p)n−2+2[1−(n−1)p(1−p)n−2].
  • Simplifying:
E[π(Brunette)]=2−(n−1)p(1−p)n−2.E[\pi(\text{Brunette})] = 2 - (n-1)p(1-p)^{n-2}.E[π(Brunette)]=2−(n−1)p(1−p)n−2.

Derivation (Best Response Analysis)

  • If the probability that another player chooses Blonde is low, being the unique player on Blonde is attractive.
  • If the probability that another player chooses Blonde is high, congestion on Blonde makes Brunette more attractive.
E[π(Brunette)]=E[π(Blonde)]E[\pi(\text{Brunette})] = E[\pi(\text{Blonde})]E[π(Brunette)]=E[π(Blonde)]
  • In a symmetric mixed equilibrium, each player must be indifferent:
3(1−p)n−1=2−(n−1)p(1−p)n−2.3(1-p)^{n-1} = 2 - (n-1)p(1-p)^{n-2}.3(1−p)n−1=2−(n−1)p(1−p)n−2.
  • Rearranging:
(1−p)n−2(3+(n−4)p)=2.(1-p)^{n-2}\bigl(3 + (n-4)p\bigr) = 2.(1−p)n−2(3+(n−4)p)=2.

Derivation (Nash Equilibrium)

  • Asymmetric pure equilibria exist when exactly one player chooses Blonde and all others choose Brunette.
  • In that profile:
    • the Blonde player does not want to switch because 3>13>13>1,
    • each Brunette player does not want to switch because switching would create congestion and reduce payoff from 111 to 000.
  • A symmetric mixed equilibrium is obtained by solving the indifference condition above.

3−2p=2⇒p=12.3-2p = 2 \quad \Rightarrow \quad p=\frac{1}{2}.3−2p=2⇒p=21​.

Nash Equilibrium

Result:

The game has:

  • asymmetric pure Nash equilibria where exactly one player chooses Blonde,
  • a symmetric mixed equilibrium where, in the strategy order (Blonde, Brunette), each player uses the probability vector (p,1−p)(p, 1-p)(p,1−p), so the equilibrium profile is
{(p,1−p),(p,1−p),…,(p,1−p)},\{(p,1-p), (p,1-p), \dots, (p,1-p)\},{(p,1−p),(p,1−p),…,(p,1−p)},

with one vector for each of the nnn players and with ppp solving

(1−p)n−2(3+(n−4)p)=2.(1-p)^{n-2}\bigl(3 + (n-4)p\bigr) = 2.(1−p)n−2(3+(n−4)p)=2.

Social Optimum

  • If nobody chooses Blonde, each player gets 222, so total welfare is 2n2n2n.
  • If exactly one player chooses Blonde, total welfare is 3+(n−1)⋅1=n+23+(n-1)\cdot 1 = n+23+(n−1)⋅1=n+2.
  • For n≥3n\geq 3n≥3, 2n>n+22n>n+22n>n+2, so the welfare-maximising outcome is that everyone chooses Brunette.

Insights

Insight:

  • The equilibrium problem is driven by congestion on the salient action.
  • The film intuition does not imply that one player approaching Blonde is socially optimal.
  • Mixed strategies capture uncertainty about who will try for the salient option.
