Game theory analysis: Signalling Game.

1 May 2026
  • Content map: SMU H3 Game Theory Map

Setup

Definition:

Signalling Game

  • Players: Two players, Student and Employer.
  • Strategies: The student chooses the number nnn of hard courses on the transcript; The employer offers wages based on the observed transcript.

Rules

  • Start with a student type privately known to the student; Players move sequentially: the student chooses hard courses and the employer chooses wages after observing the transcript.
  • The player who uses the signal or wage rule optimally maximises payoff.
  • There are two student types:
    • type AAA: intrinsically motivated,
    • type CCC: grade oriented.
  • Employer values type AAA at 160,000160{,}000160,000 and type CCC at 60,00060{,}00060,000; Outside options are 125,000125{,}000125,000 for type AAA and 30,00030{,}00030,000 for type CCC.
