gpu-watch

A lightweight CLI wrapper for long-running GPU jobs — persistent logs, GPU telemetry, metric extraction, and push notifications.

What it does

Works with any training command. No code instrumentation required.

📁 Persistent timestamped run directories 📝 Live log streaming via FIFO + tee 📊 GPU telemetry via nvidia-smi 📈 Multi-source metric extraction 🔔 ntfy.sh push notifications 🔢 Exit code propagation ⚡ SIGINT + SIGTERM handling 🔄 Multi-phase sequential training 🌐 Kubernetes-aware 🖥️ Works on RunPod, cloud VMs, local
Quick install (Debian/Ubuntu)
# clone and install git clone https://github.com/horse-3903/gpu-watch.git
cd gpu-watch
sudo make install

# wrap any training command gpu-watch --topic my-topic -- python train.py
